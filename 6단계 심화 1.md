# 2444. 별 - 7
## 문제: 예제를 보고 규칙을 유추한 뒤에 별을 찍어 보세요.
## 예제 입력: 5
## 예제 출력: 
```
    *
   ***
  *****
 *******
*********
 *******
  *****
   ***
    *
```
## 문제풀이
```
const n = require('fs').readFileSync('/dev/stdin');
let result = '';
                     
for(let i = 1; i <= n * 2 - 1; i++){
    i <= n ? result += ' '.repeat(n - i) + '*'.repeat(i * 2 - 1)
           : result += ' '.repeat(i - n) + '*'.repeat((n * 4) - (i * 2) - 1)      
    if(i !== n*2-1) result += '\n'
}
console.log(result);
```

# 4344. 평균은 넘겠지
## 문제: 대학생 새내기들의 90%는 자신이 반에서 평균은 넘는다고 생각한다. 당신은 그들에게 슬픈 진실을 알려줘야 한다.
## 문제풀이
```
const [n, ...arr] = require('fs').readFileSync('/dev/stdin').toString().trim().split('\n');
console.log(arr.map(v => {
    const [c, ...scores] = v.split(' ').map(Number);
    const avg = scores.reduce((acc, cur) => acc + cur, 0) / scores.length; // 학생 평균
    const rate = scores.filter(v => v > avg).length / scores.length * 100; // 평균을 넘는 학생의 비율
    return `${rate.toFixed(3)}%`; // 소수 셋째자리까지 리턴
}).join('\n')
);
```
